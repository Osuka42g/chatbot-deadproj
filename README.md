# lazy-ferret
Show me the ferrets! Custom bot to determinate if given picture contains a ferret.

Requisites:
- [ngrok](https://ngrok.com/)
- Facebook Fanpage
- Facebook APP
- Google Cloud API Key
- Go 1.9

### Structure of the project
./messaging
./middleware
./google-vision
main.go
